Основной проект

Сайт типографии tipkor.ru

Cайт для небольшой типографии. Предоставляет кроме статичной информации (контакты, карта, описание) еще модули расчета заказов по 3м разным направлениям деятельности компании. Производится расчет цены и сроков готовности продукции. После расчет есть возмоность оформить заказ. Клиент получает письмо. Вся информация по заказу, расчету, клиенту записывается в базу данных.
